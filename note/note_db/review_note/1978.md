### <br>내 풀이 => "맞았습니다."
```python
import sys
input = sys.stdin.readline

N = int(input())
num = list(map(int, input().rstrip('\n').split(' ')))

result = 0
for i in num:
    count = 0
    for j in range(0, i):
        if i % (j+1) == 0:
            count += 1
    if count == 2:
        result += 1

print(result)
```

### <br>인터넷 참고 풀이 2
```python
sieve = [True] * 1001
sieve[0], sieve[1] = [False] * 2

i = 2
while i <= 1000:
    if sieve[i]:
        for t in range(i+i, 1001, i):
            if sieve[t]:
                sieve[t] = False
    i += 1

N = int(input())
num = list(map(int, input().rstrip('\n').split(' ')))

result = 0
for i in num:
    if sieve[i]:
        result += 1

print(result)
```
